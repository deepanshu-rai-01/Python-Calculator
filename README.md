# 🧮 Python Calculator

A menu-driven calculator built using Python that performs multiple mathematical operations with proper input validation and exception handling.

## 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Power Calculation
- Floor Division
- Input Validation
- Division by Zero Handling
- Menu-Driven Interface

## 🛠 Technologies Used

- Python

## ▶️ How to Run

1. Download the project.
2. Open terminal or command prompt.
3. Run:

```bash
python calculator.py
```
